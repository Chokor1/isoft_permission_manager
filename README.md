# Isoft Permission Manager

Delegated permission management for Frappe / ERPNext.

A System Manager defines **delegations**: which users (Managers) are allowed to
modify permissions, and exactly what they may touch — which target users, which
roles and which modules. Managers (users with the **ISOFT Permission Manager**
role) then use a single smart view to filter users and manage their roles, user
permissions and module access, all within the scope granted to them.
